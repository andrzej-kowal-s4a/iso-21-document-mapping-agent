# Selected Documents for ISO 27001 Control 5.1 Policies for information security

## 1. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

This is the primary information security policy document that directly addresses the control requirement. It defines the organization's high-level approach to managing information security and is approved by top management (Chief Information Officer), making it the most critical document for this control.

## 2. Policy, Standards, and Procedures Exceptions Procedure
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170107067

This procedure directly supports the control requirement for "procedures for handling exemptions and exceptions" as mentioned in the ISO 27001 guidance. It establishes how the organization manages exceptions to its information security policies.

## 3. Policy Owner vs Author of procedures, standards, guidances and processes
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170102632
