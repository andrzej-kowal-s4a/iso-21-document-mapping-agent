# Selected Documents for ISO 27001 Control 5.7

## 1. Vulnerability Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/214794244

This policy directly addresses the collection and analysis of information about existing or emerging threats, which is essential for producing threat intelligence. It outlines the organization's approach to identifying, assessing, and managing vulnerabilities that could be exploited by threats.

## 2. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

The Information Security Policy provides the overarching framework for information security management, which includes requirements for threat intelligence collection and analysis. It establishes the foundation for implementing control 5.7 by setting the tone and expectations for security practices across the organization.

## 3. Security Incident Reporting Procedure / Procedura zgłaszania incydentów związanych z bezpieczeństwem
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104051

This procedure is relevant as it describes how security incidents are reported and handled, which is closely related to the collection of threat intelligence. It ensures that information about actual or emerging threats is captured and analyzed as part of the organization's threat intelligence activities.
