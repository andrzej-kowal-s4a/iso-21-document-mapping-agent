# Selected Documents for ISO 27001 Control 7.3 Securing offices, rooms and facilities

## 1. Procedura korzystnia z biura po godzinach pracy / After-Hours workplace activity procedure
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104529

This procedure directly addresses physical security controls for office facilities, including guidelines for working after hours, building access protocols, and requirements for securing offices when the last employee leaves. It ensures physical security perimeter control and authorized access management outside normal business hours.

## 2. Office Alarm Code
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170107751

This procedure establishes critical physical security controls through alarm system management. It defines who can obtain alarm codes (6-month employment requirement), proper arming/disarming procedures, and incident reporting requirements. This directly supports preventing unauthorized physical access to offices and facilities.

## 3. Alarm codes management
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4AIH/pages/149520699

This internal helpdesk procedure details the technical process for managing alarm system access codes, including creating, modifying, and removing employee access codes. It provides essential controls for managing physical access authorization and ensuring only authorized personnel can access secured facilities.

## 4. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

This umbrella policy includes a dedicated Physical Security section that establishes high-level requirements for securing information processing and storage facilities. It mandates security perimeters, appropriate security barriers, access controls, environmental protection, and visitor escort requirements - all core elements of Control 7.3.

## 5. Employee Offboarding
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4AIH/pages/149520666

This procedure ensures physical security controls are maintained when employees leave by requiring removal of alarm codes, return of access cards, and revocation of building access rights. It prevents unauthorized physical access by former employees and supports maintaining the integrity of facility security perimeters.

## 7. Information Classification Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170101881

This policy supports physical security by establishing information sensitivity levels that determine which information requires physical protection measures. It provides the foundation for understanding what information and assets in offices and facilities need to be protected from unauthorized physical access, damage, and interference.
