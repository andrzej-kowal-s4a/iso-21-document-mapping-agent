# Selected Documents for ISO 27001 Control 8.30 Outsourced development

## 1. Supplier Security Risk Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170099594

This policy is the most directly relevant document as it comprehensively addresses security requirements for suppliers and outsourced services. It covers contractual security requirements, security assessments, monitoring of supplier security practices, and includes template security clauses - all key elements required by ISO 27001 Control 8.30 for managing outsourced development security.

## 2. Vulnerability Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/214794244

This policy is essential for outsourced development as it defines how vulnerabilities are identified, evaluated, and treated in software products. It addresses the control's requirement for provision of evidence that sufficient testing has been applied to guard against known vulnerabilities and ensures minimum acceptable security levels are maintained in outsourced deliverables.

## 3. Procedure - Service transition to prod
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/SUP/pages/160433255

This comprehensive procedure covers acceptance testing for quality and accuracy of deliverables, including verification of security requirements, monitoring setup, and knowledge transfer. It directly supports the control's requirements for directing, monitoring, and reviewing outsourced system development activities and ensuring security measures are implemented before production deployment.

## 4. Change Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104688

This policy establishes the formal process for managing changes to IT systems and services, which is critical for outsourced development. It ensures that all changes from external suppliers follow structured planning, evaluation, review, approval, and documentation processes - supporting the control's requirement to direct and monitor outsourced development activities.

## 5. Legal and Regulatory Compliance Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170106479

This policy addresses the control's requirement to take consideration of applicable legislation when managing outsourced development. It ensures that Smart4Aviation and its suppliers comply with all applicable laws and regulations, including those related to personal data protection and other regulatory requirements relevant to outsourced system development.

## 6. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

This umbrella security policy establishes the foundational security requirements that apply to all systems, including those developed by external parties. It covers secure SDLC, security testing, code security practices, and system security controls - all essential for ensuring outsourced development meets organizational security standards as required by the control.

## 7. Internal Auditing Policy for Security
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/2086699019

This policy supports the control's requirement for contractual right to audit development processes and controls. It establishes the framework for conducting security audits, evaluating compliance with ISO 27001 requirements, and ensuring that security measures in outsourced development are properly implemented and maintained through regular verification.

## 8. Encryption and Cryptography Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170105815

This policy is relevant to outsourced development as it defines encryption requirements for data at rest and in transit, key management, and cryptographic standards. It ensures that external developers implement appropriate data protection measures and helps establish minimum acceptable levels of security capabilities in outsourced deliverables as required by the control.
