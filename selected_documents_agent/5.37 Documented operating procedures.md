# Selected Documents for ISO 27001 Control 5.37 Documented operating procedures

## 1. Server management procedure
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170107313

This comprehensive procedure documents the complete lifecycle of server management including creation requests, environment preparation, resource allocation, backup procedures, and decommissioning. It directly addresses the control's requirements for documented operating procedures for information processing facilities, including responsible individuals, system configuration, scheduling requirements, and restart/recovery procedures.

## 2. Procedure - Service transition to prod
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/SUP/pages/160433255

This procedure provides detailed operational steps for transitioning services to production, covering system installation, configuration validation, backup procedures, monitoring setup, error handling, and escalation contacts. It comprehensively addresses multiple aspects of the control including secure configuration, backup requirements, monitoring procedures, and support contacts for operational difficulties.

## 3. New NAGIOS monitoring request procedure
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104450

This procedure documents the standardized process for implementing monitoring of information processing facilities, including monitoring thresholds, reaction procedures, responsibility assignments, and escalation contacts. It directly supports the control's requirements for monitoring procedures and instructions for handling exceptional conditions during system operation.
