# Selected Documents for ISO 27001 Control 5.6 Contact with special interest groups

## 1. Vulnerability Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/214794244

This policy directly supports the control's requirement to receive early warnings, alerts, advisories, and patches about vulnerabilities. It establishes processes for identifying, evaluating, and treating vulnerabilities discovered through various sources including external vulnerability databases and security information, which aligns with maintaining contact with security forums and specialist groups.

## 2. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

As the umbrella security policy, this document establishes the overall framework for information security management including security governance, risk management, and incident management. It provides the foundational authority for maintaining relationships with external security groups and forums to stay current with security threats and best practices, supporting all aspects of Control 5.6.

## 3. Security Incident Reporting Procedure
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104051

This procedure directly relates to Control 5.6's guidance point (f) regarding providing suitable liaison points when dealing with information security incidents. It establishes formal processes for incident reporting and management, which requires coordination with external parties and security forums during incident response activities.

## 4. Supplier Security Risk Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170099594

This policy demonstrates engagement with external parties to share and exchange information about security practices, threats, and vulnerabilities. It includes processes for security assessments, RFI questionnaires, and ongoing communication with suppliers about security requirements, which supports the information exchange aspects of Contact with special interest groups.

## 5. Legal and Regulatory Compliance Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170106479

This policy ensures the organization stays current with applicable laws and regulations, which requires maintaining awareness of regulatory developments through professional associations and regulatory forums. It directly supports the control's objective of ensuring understanding of the information security environment remains current through external contacts.

## 6. Training Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170106783

This policy supports the control by enabling personnel to attend conferences, certifications, and external training that facilitate knowledge sharing with professional associations and special interest groups. It provides the framework for improving knowledge about best practices and staying up to date with relevant security information through external professional development opportunities.
