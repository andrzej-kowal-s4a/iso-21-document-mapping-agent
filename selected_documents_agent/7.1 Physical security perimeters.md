# Selected Documents for ISO 27001 Control 7.1 Physical Security Perimeters

## 1. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

This overarching security policy establishes the foundation for all security controls including physical security. It specifically addresses physical security requirements such as security perimeters, physical barriers, access controls, visitor escorting, and protection of information processing facilities, making it essential for demonstrating compliance with ISO 27001 physical security requirements.

## 2. Alarm codes management
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4AIH/pages/149520699

This procedure directly addresses the management of alarm systems and access codes for physical security perimeters. It details the process for adding and removing alarm codes for employees, which is a key control mechanism for monitoring and controlling access to secured areas, directly supporting the requirement for alarming, monitoring and testing security perimeters.

## 3. Office Alarm Code
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170107751

This procedure provides detailed instructions for obtaining and using alarm codes to arm and disarm the office security system. It addresses physical security perimeter protection by defining how employees access secured areas, when alarms must be activated, and how to respond to alarm activations. This directly supports the control's requirement for appropriate security barriers and access controls.

## 4. Procedura korzystnia z biura po godzinach pracy / After-Hours workplace activity procedure
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104529

This procedure governs physical access and security management when working outside core hours, including requirements for alarm codes, visitor restrictions, and security responsibilities. It addresses physical security perimeter controls for after-hours access, ensuring that security measures are maintained even during non-standard working hours, which is critical for continuous protection of facilities.

## 5. Employee Offboarding
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4AIH/pages/149520666

This procedure includes critical physical security perimeter controls by ensuring that departing employees have their physical access rights revoked, alarm codes deactivated, and access to secured areas removed. This directly supports the control requirement for managing access to physical security perimeters and preventing unauthorized physical access by former personnel.
