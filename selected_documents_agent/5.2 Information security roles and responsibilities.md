# Selected Documents for ISO 27001 Control 5.2 Information security roles and responsibilities

## 1. Information Security Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104722

This document establishes the foundation for information security governance within the organization, defining the overall security framework under which roles and responsibilities are allocated. It directly supports the control by providing the high-level structure for implementing and managing information security.

## 2. Security Incident Reporting Procedure / Procedura zgłaszania incydentów związanych z bezpieczeństwem
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104051

This procedure defines responsibilities for reporting and handling security incidents, which aligns with the control's requirement to establish clear roles for information security processes. It specifies who is responsible for incident management, response, and escalation pathways within the organization.

## 3. Risk Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170100789

This policy directly addresses the control requirement for information security risk management activities and acceptance of residual risks. It establishes the roles and responsibilities related to risk assessment, treatment, and ownership, which is specifically mentioned in guidance point (c) of the control.

## 4. Change Management Policy
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/170104688

The policy outlines roles and responsibilities for managing changes to systems and processes, addressing the control's requirement for protection of information assets and carrying out specific security processes. It establishes accountability for changes that could affect information security within the organization.

## 5. Internal Auditing Policy for Security
**Confluence URL:** https://smart4aviation.atlassian.net/wiki/spaces/S4A/pages/2086699019

This document defines roles and responsibilities related to security auditing, which supports the control by establishing a structured approach to verify that security responsibilities are properly assigned and fulfilled. It helps ensure that delegated security tasks are correctly performed as mentioned in the control guidance.
