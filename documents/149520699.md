This is internal helpdesk procedure connected with alarm code management

1. Log in to console.  
   Use your personal alarm code to log in and confirm by pressing \*.
2. Go to "użytkownicy" tab by using up/down arrows.
3. Delete code: go to "usuwanie" tab, find employee you want to remove and press "ok" ATTN! pressing ok = delete without any confirmation and possiblity to restore.
4. Please ask employee to provide you three six digit (minimum) codes he could use
5. Creating new account:  
   Go to "nowy użytkownik" tab:  
   Set password: 6 digits minimum and other than any code in base\*   
    \* In case first entered code is same as already existing one in alarm console - console will not allow to use it again - please try other one from three provided from employee  
   "Strefy" tab: here you can add person rights to different floors, use 0 to mark/unmark floors  
   "Uprawnienia" tab: mark first 3, unmark rest, use 0 to mark/unmark floors  
   "Nazwa" tab: use keyboard to fill in employee name and surname (up arrow works like backspace)

Press \* to confirm all changes and \* again to log out from console.
